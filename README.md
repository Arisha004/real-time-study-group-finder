# Real-Time Study Group Finder

## Overview
A real-time platform where students can find and join study groups based on their courses, available times, campus location, and preferred study style. Features live chat, shared resources, and group scheduling.

## Tech Stack
- React
- Supabase
- WebSocket
- Tailwind CSS

## Architecture
- **Frontend**: React + Vite
- **Backend**: Node.js (Express)
- **Database**: Supabase

## Quick Start

### Prerequisites
- Node.js 18+



### Installation
```bash
# Frontend
cd frontend && npm install

# Backend
cd backend && npm install

# Environment
cp backend/.env.example backend/.env
# Edit .env with your credentials

# Database setup



# Run
cd backend && npm run dev
# In another terminal:
cd frontend && npm run dev
```
